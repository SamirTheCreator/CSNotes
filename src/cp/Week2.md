# Week 2

## Time complexity

Useful time complexity assumptions by the size of the given input.

| Size of the input      | Expected time complexity |
| -----------            | ----------- |
| n <= 10 | O(n!) |
| n <= 20 | O(2<sup>n</sup>) |
| n <= 500 | O(n<sup>3</sup>) |
| n <= 5000 | O(n<sup>2</sup>) |
| n <= 10<sup>6</sup> | O(nlogn) or O(n) |
| n is large | O(logn) or O(1) |
